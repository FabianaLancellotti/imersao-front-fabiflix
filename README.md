🎬 Fabiflix - Interface Estilo Netflix
Este projeto é uma aplicação web interativa que replica a experiência de usuário da Netflix, desenvolvida durante a Imersão DEV com Google Gemini. O foco principal foi a criação de uma interface moderna, responsiva e funcional, utilizando técnicas avançadas de front-end.

🚀 Tecnologias Utilizadas
* HTML5: Estruturação semântica da página de seleção de perfis e catálogo.
* CSS3 (Advanced): Uso de Variáveis CSS para implementação de Dark/Light Mode, Flexbox/Grid para layout e Media Queries para total responsividade em dispositivos móveis.
* JavaScript (ES6): Manipulação dinâmica do DOM, persistência de dados com LocalStorage para salvar a preferência de tema e o perfil selecionado, e interatividade com trailers.

🛠️ Funcionalidades
* Seleção de Perfis: Interface intuitiva onde o usuário escolhe quem está assistindo, com persistência do nome e imagem do perfil.
* Troca de Tema Dinâmica: Implementação de um switch para alternar entre modo claro e escuro, salvando a escolha do usuário no navegador.
* Catálogo Interativo: Fileiras de filmes com efeito de zoom no hover, exibição de metadados (score, classificação, duração) e integração com vídeos.
* Design Responsivo: A interface se adapta perfeitamente de monitores UltraWide a smartphones pequenos.

🧠 Evolução e Aprendizado
Mais do que a aplicação de ferramentas técnicas, este projeto representa um marco na minha evolução intelectual e transição de carreira para Dados e BI. Durante o desenvolvimento, exercitei:

* Lógica de Programação: Estruturação de scripts para gerenciamento de estados de interface.
* User Experience (UX): Foco na jornada do usuário, desde a entrada até a navegação no conteúdo.
* Resolução de Problemas: Troubleshooting de comportamentos de CSS e carregamento dinâmico de elementos.
